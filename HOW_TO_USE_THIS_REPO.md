# How To Use This Repo

This repo is only the roadmap hub.

Do not put every full ML project inside this repo.

Instead:

1. Pick the next project from the roadmap.
2. Create a new separate GitHub repository for that project.
3. Copy the contents of `templates/project-template/` into the new repo.
4. Build the project there.
5. Come back to this roadmap repo.
6. Add the finished repo link to `README.md` and `progress-tracker.csv`.

Example:

Roadmap repo:

```text
ml-roadmap
```

Separate project repo:

```text
linear-regression-house-prices
```

The roadmap tracks progress.  
The project repos show the actual work.
