# ML Roadmap

This repo is my central roadmap for learning machine learning.

It is not where all my full projects will live.

Instead, this repo tracks:

- the models I plan to build
- the order I will learn them in
- the skills each project teaches
- links to the separate GitHub repos for each finished project
- my progress over time

Each serious ML project will get its own separate GitHub repository.

---

## Why This Repo Exists

The goal is to build a strong ML portfolio step by step.

Instead of randomly making projects, this roadmap gives me a clear path:

1. Learn the model.
2. Build a small working project.
3. Document what I learned.
4. Push the full project to its own GitHub repo.
5. Link that repo here.

---

| Level | Focus | Goal |

## How I Will Build Each Project

Each separate project repo should contain:

```text
project-name/
  README.md
  notebook.ipynb
  src/
  data/
  requirements.txt
```

Each project README should explain:

- what the model does
- what dataset I used
- how the model works
- how I cleaned the data
- how I trained the model
- what metrics I used
- what results I got
- what I learned
- what I would improve next
